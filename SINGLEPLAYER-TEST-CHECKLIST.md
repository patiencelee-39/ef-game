## 🐛 常見問題排查

### 如果遊戲卡住

1. 開啟開發者工具（F12）
2. 查看 Console 錯誤訊息
3. 查看 Network 是否有載入失敗
4. 重新整理頁面（Ctrl+R / Cmd+R）

### 如果刺激物不顯示

1. 檢查 Console 是否有 SVG 相關錯誤
2. 檢查 `svg-assets.js` 是否正確載入

### 如果工作記憶測試不出現

1. 檢查 Console 是否有錯誤
2. 檢查 `game-logic-wm.js` 是否正確載入
3. 查看之前的修復是否生效

### 如果音效不播放

1. 檢查瀏覽器是否靜音
2. 檢查瀏覽器是否允許自動播放音訊
3. 嘗試點擊畫面後再測試
