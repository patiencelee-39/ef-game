# 📖 故事對話文字檔

這個資料夾包含 12 個探險點的對話文字。

## 給教師的說明

1. 每個 `.txt` 檔案對應一個探險點
2. 用任何文字編輯器（記事本、VS Code）打開即可修改
3. 修改完成後，請執行重新產生指令（詳見下方）

## 檔案格式說明

每個 `.txt` 檔案包含三段對話，用 `---` 分隔：

```
[開場] 角色名稱
對話文字...

---

[通過] 角色名稱
對話文字...

---

[失敗] 角色名稱
對話文字...
```

### 角色名稱對照表

| 寫法             | 角色               | 圖示 |
| ---------------- | ------------------ | ---- |
| `mentor`         | 智慧貓頭鷹「阿智」 | 🦉   |
| `villain`        | 搗蛋狐狸「淘淘」   | 🦊   |
| `legendaryEagle` | 金鷹王者           | 🦅   |

## 重新產生遊戲設定

修改 `.txt` 檔案後，在專案根目錄執行：

```bash
node tools/build-stories.js
```

這會自動更新 `js/story-config.js` 中的對話內容。
