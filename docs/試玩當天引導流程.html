🎮 兒童試玩當天 — 引導流程手冊

執行功能訓練遊戲 ｜ 個別施測版（1-3 位兒童）

📋 流程總覽

總時長
25–35 分鐘
階段數
7 個階段
適用對象
3–6 歲
設備
平板 / 電腦
📦 Phase 0｜事前準備（孩子到場前） 提前 15 分鐘

✅ 準備清單

若在機構進行，先與現場教師確認孩子的慣用名/暱稱，方便建立親近感。若為聽障兒童，確認其主要溝通方式（口語、手語、混合）。
👋 Phase 1｜暖場與建立關係 約 3 分鐘

目標：讓孩子放鬆、建立信任，降低陌生焦慮。

  1. 蹲下到與孩子平視的高度，微笑打招呼。
  2. 簡單自我介紹，詢問孩子的名字（或使用已知暱稱）。
  3. 聊 1–2 句輕鬆話題（例如：今天吃了什麼？喜歡什麼動物？）。
  4. 介紹今天要做的事情（用「玩遊戲」而非「測驗」來描述）。
「嗨！我是珮瑄姐姐～你叫什麼名字呀？（等待回應）好棒！今天姐姐帶了一個很有趣的遊戲要跟你一起玩喔！裡面有小老鼠、起司、還有小魚耶！你想不想看看？」
不要催促孩子。如果孩子害羞或緊張，多花一些時間暖場，讓家長/老師先陪在旁邊也可以。
📖 Phase 2｜遊戲示範與規則教學 約 5 分鐘

目標：用孩子能理解的方式說明遊戲規則，確認孩子理解後才開始。

Step 1：進入遊戲

  1. 點選「📱 單人模式」→ 進入探險地圖。
  2. 指著地圖第一關（🐭 小老鼠 × 規則一）跟孩子說：
「你看！這是一張冒險地圖，我們要一關一關地闖喔！先從這一關開始！」

Step 2：用示範動畫教規則

遊戲內建示範動畫（Plan A），會自動用 5 個階段展示規則。引導者配合動畫口語說明：

動畫階段 畫面內容 引導者口語配合
1. 準備 「注意看喔！」提示 「眼睛看好喔～」
2. 出現刺激物 🧀 起司出現 + 名字標籤 「你看！是起司耶！」
3. 指示動作 綠色指示「看到🧀起司 → 按按鈕！」
按鈕會跳動		 「看到起司就要趕快按這個按鈕喔！」
4. 模擬動作 按鈕被按下 → 答對回饋 「按對了！好棒！出現 ✓ 就是答對喔！」
5. No-Go 示範 😺 貓出現 →「看到😺貓咪 → 不要按！」 「看到貓咪的時候，手手不要動喔！因為貓咪會把起司吃掉～」

Step 3：練習試驗（Plan D）

示範結束後自動進入 3 題練習：

  1. 讓孩子自己操作，觀察是否理解規則。
  2. 如果按錯，遊戲會自動顯示提示（例如：「😺 貓咪 → 不要按喔！」）。
  3. 引導者可在旁邊口頭鼓勵，但不要幫孩子按。
「現在換你自己試試看！記得喔：看到起司就按，看到貓咪就不要按～」
如果孩子練習 3 題後仍不太理解，可以再跑一次示範動畫（遊戲會提供重看選項）。最多重複 2 次示範即可開始正式遊戲，避免孩子失去興趣。
🎮 Phase 3｜正式遊戲進行 約 12–18 分鐘

目標：讓孩子按自己的節奏完成探險地圖關卡，引導者觀察並記錄。

遊戲進行順序（探險地圖）

關卡 內容 預估時間 引導重點
① 🐭 規則一 起司按、貓不按（6 題） 2–3 分鐘 基礎規則建立，多鼓勵
② 🐭 規則一+WM 規則一 + 位置記憶 3–4 分鐘 第一次出現 WM，需額外說明
③ 🐭 規則二 起司不按、貓按（規則反轉） 2–3 分鐘 關鍵轉折！需提醒規則改變
④ 🐭 規則二+WM 規則二 + 位置記憶 3–4 分鐘 觀察適應程度
⑤ 🐭 混合 沒人→規則一、有人→規則二（12 題） 4–5 分鐘 最具挑戰！注意情境線索
⑥ 🐭 混合+WM 混合 + 位置記憶 5–6 分鐘 最高難度，適時鼓勵
試玩當天不一定要全部完成。建議至少完成前 3 關（①②③），讓孩子體驗「規則建立→規則反轉」的核心機制。如果孩子狀態好，可以繼續往後闖關。

引導原則

  1. 鼓勵為主：「好棒！」「你好厲害！」「沒關係，再試一次！」
  2. 不主動介入：讓孩子自己操作，即使按錯也不要馬上糾正（遊戲會提供回饋）。
  3. 適時提醒核心規則：進入新關卡時，用簡短話語提醒。
  4. 觀察記錄：在觀察表上記錄孩子的表情、口語反應、猶豫點、求助行為。
  5. 尊重孩子意願：如果孩子表示不想繼續，不強迫，可以休息後再嘗試或結束。

各關卡轉場口語提示

進入規則二（③）時：「接下來喔，規則要變了！以前看到起司要按對不對？但是這一關不一樣喔～看到起司反而不能按，看到貓咪才要按！因為現在起司已經被藏起來了，貓咪變成好朋友囉！」
進入混合（⑤）時：「哇！這一關更特別喔！你要注意看畫面上有沒有出現人喔。沒有人的時候，就跟第一關一樣 — 看到起司就按。但如果有人出現，就要反過來 — 看到貓咪才按！要注意看有沒有人出現喔！」
遇到 WM 附加環節（②④⑥）時：「等一下玩完之後，會出現一個記憶小遊戲喔！螢幕上面的格子會一個一個亮起來，你要記住亮了哪些，然後照順序按出來。跟記路線一樣！」
如果孩子在某一關連續失敗情緒低落，可以暫時跳過或回到已通過的關卡重玩，讓孩子恢復信心。遊戲支援重玩已通過的關卡。
☕ Phase 4｜彈性休息（視需要） 約 2–3 分鐘

目標：根據孩子的專注力與情緒狀態，適時安排休息。

  1. 觀察訊號：孩子開始東張西望、摸摸臉、身體扭動、嘆氣、反應變慢 → 需要休息。
  2. 自然轉場：在關卡之間穿插休息，而非遊戲中途打斷。
  3. 休息活動：喝水、伸懶腰、聊聊剛剛玩到什麼、看看窗外。
  4. 復歸確認：休息後先問「準備好了嗎？我們繼續冒險吧！」。
3–4 歲建議每 2 關休息一次；5–6 歲通常可以連續完成 3–4 關。依孩子個別差異彈性調整。遊戲支援暫停，進度不會消失。
🎉 Phase 5｜體驗結束與回饋收集 約 3–5 分鐘

目標：收集孩子的主觀感受反饋，正向結束體驗。

Step 1：正向收尾

「你今天好棒喔！闖了好多關！你最喜歡哪一個部分？覺得哪裡最好玩？」

Step 2：簡易回饋問題（用臉譜 / 手勢回答也可以）

# 問題 回答方式
1 你覺得這個遊戲好玩嗎？ 😊 很好玩 / 😐 還好 / 😞 不好玩
2 哪個部分最好玩？ 開放回答（記錄原話）
3 有沒有哪裡太難了？ 開放回答
4 畫面上的東西你都看得懂嗎？ 是 / 哪裡看不懂
5 你還想再玩嗎？ 😊 想！/ 😐 嗯 / 😞 不想

Step 3：致贈小禮物

「謝謝你今天幫姐姐一起玩遊戲！你幫了好大的忙喔！這是送給你的小禮物，謝謝你～」

Step 4：與家長簡短溝通（若家長在場）

  1. 簡短分享孩子的表現亮點（例如：「他在規則切換那邊適應得很快」）。
  2. 如果家長有興趣，可以介紹遊戲網址讓他們回家繼續玩。
  3. 提醒遊戲是免費的學術研究工具，歡迎繼續使用。
📝 Phase 6｜資料整理（孩子離開後） 約 5–10 分鐘

目標：趁記憶鮮明時儘速完成觀察紀錄與資料匯出。

  1. 匯出成績 CSV：進入成績結算頁面 → 點選「📤 匯出 CSV」→ 儲存檔案。
  2. 截圖成績畫面：使用遊戲內建截圖功能或手動截圖。
  3. 完成觀察紀錄表：補齊剛才來不及記錄的觀察內容。
  4. 停止錄音並備份：以匿名編碼命名（例：P01_20260301.m4a）。
  5. 清除裝置資料：設定 → 清除資料（為下一位受試者準備）。
  6. 回顧重點：孩子在哪些關卡表現特別好/差？哪些 UI 讓孩子困惑？
建議當天結束後統一整理所有受試者的觀察紀錄，彙整成結構化的改善建議清單。

規則理解度： 示範後是否能掌握？需要幾次示範？練習試驗的表現？
規則切換適應： 進入規則二/混合時的反應？切換困難程度？
操作順暢度： 觸控/鍵盤操作是否順利？有無按錯位置？
