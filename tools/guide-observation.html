📖 兒童試玩當天 — 引導流程手冊

試玩當天的完整標準化流程，點擊各階段展開詳細步驟。

📋 階段一：事前準備（測試前 30 分鐘）
  • ✅ 設備已充電，確認電量足夠完成測試
  • ✅ Wi-Fi 連線穩定，開啟遊戲網站確認載入正常
  • ✅ 將瀏覽器設為全螢幕模式（避免兒童誤觸網址列）
  • ✅ 音量調至適中（遊戲有語音提示）
  • ✅ 準備觀察紀錄表（可用下方數位版或紙本）
  • ✅ 場地安靜、光線充足、桌椅高度適合兒童
  • ✅ 準備獎勵貼紙或小禮物

QR code 張貼確認

  • ✅ 前方螢幕旁：投影顯示，或 A4 紙貼於螢幕旁
  • ✅ 左側牆壁：A3 以上尺寸，高度約 130–150 cm（家長視線平行）
  • ✅ 右側牆壁：同上
  • ✅ 短網址可正常開啟，量表頁面載入正常
  • ✅ 兒童代號已指定（如 C001），記錄於觀察紀錄表
💡 QR code 紙張建議標示：
「【請家長填寫】孩子玩遊戲前，請先掃描填寫一次；孩子玩完後，請再掃描填寫一次。請在第一題填入施測者給您的『兒童代號』（例如：C001）」
🤝 階段二：接待與暖身＋家長前測（5~10 分鐘）
  • 親切迎接兒童與家長／老師
  • 確認知情同意書已簽署

向家長說明流程（統一話術）

「歡迎！今天孩子會來玩一個執行功能訓練遊戲，全程大約 45 分鐘。在孩子開始玩之前，麻煩您先用手機掃描牆上的 QR code，填寫一份簡短問卷。問卷裡第一題會請您填入一組代號，代號是『C001』（依實際代號告知），請記住這個代號，等一下填第二次也需要用到。所有資料都會匿名處理。」

TC-CHEXI 量尺說明（口頭提示家長）

1 = 從不孩子幾乎不出現此行為
2 = 很少偶爾出現
3 = 有時約一半時間出現
4 = 經常大多數時間出現
5 = 總是幾乎每次都出現
⚠️ 前測必須在兒童開始遊戲之前完成，不可跳過。確認方式：口頭詢問「您填完了嗎？」，並記錄前測完成時間。

同步進行：與兒童暖身

  • 家長填寫前測的同時，施測者與兒童輕鬆聊天建立信任
  • 「你好！今天我們要來玩一個有趣的遊戲，裡面有小老鼠跟起司，你有沒有看過起司呀？」
💡 如果兒童害羞，不急著開始，先讓他熟悉環境和施測者。
🎮 階段三：遊戲操作引導（主要流程）

回合開始前 — 統一指導語

「我們現在要來玩一個遊戲。你看，小老鼠很餓，牠要找起司吃。當螢幕出現起司的時候，你就趕快按這裡（示範）；如果你看到貓咪出現，就要忍住，不可以按。我們先來練習看看。」
  • 1. 協助兒童在設備前坐好，確認手指可舒適觸碰螢幕／鍵盤
  • 2. 開啟遊戲 → 首頁 → 點選「單人練習」
  • 3. 選擇遊戲關卡後，系統會詢問「要先觀看示範和做練習嗎？」
    → 第一次玩：選「觀看示範與練習」
    → 已熟悉規則：可選「跳過，直接開始」
  • 4. 規則動畫播放時，可說：「我們先來看看怎麼玩喔～」
  • 5. 練習階段：鼓勵嘗試 —「很棒！」「沒關係，再試一次！」
  • 6. 正式測驗：「現在要正式開始囉，盡量又快又對！加油！」

練習完成後 — 確認兒童理解

  • 問：「出現起司要做什麼？」→ 預期回答：按
  • 問：「出現貓咪要做什麼？」→ 預期回答：不按
💡 若兒童無法回答，可再示範一次，並記錄於觀察紀錄表。

各回合對應的執行功能向度

回合一Go/No-Go（老鼠洞）→ 抑制控制 → TC-CHEXI INHIB
回合二DCCS 標準版 → 認知彈性 → TC-CHEXI REG
回合三Go/No-Go（海洋）→ 抑制控制 → TC-CHEXI INHIB
回合四DCCS 進階版 → 認知彈性 → TC-CHEXI REG
工作記憶段順序記憶 → 工作記憶 → TC-CHEXI WM
💡 施測者在旁觀察並填寫觀察紀錄表，不介入、不提示答案。可提前提醒家長：「等一下孩子在玩的時候，請您在旁邊靜靜觀察就好，不需要提示他。」
👀 階段四：觀察重點
  • 🧠 規則理解：兒童是否能跟上規則切換？
  • ⏱️ 反應速度：按鍵是否猶豫？有無明顯變快／變慢？
  • 🎯 正確率：是否頻繁出錯？出錯時的反應？
  • 😊 情緒狀態：開心、專注、焦慮、無聊、挫折？
  • 💪 投入程度：主動想繼續還是想停下？
  • 👆 操作能力：手指協調、能否自行操作？
  • 📝 額外提示：是否需要口頭提示？提示了幾次？
💬 階段五：試玩後兒童訪談（3~5 分鐘）

遊戲結束後，以簡單問題詢問兒童（可搭配笑臉／哭臉表情卡）。

  • 「這個遊戲好不好玩？」
  • 「哪一關最難？」
  • 「你還想再玩嗎？」
💡 將兒童的原話記錄於下方觀察紀錄表的「兒童回饋摘要」欄位。
🏁 階段六：結束與數據確認（5 分鐘）
  • 遊戲結束後讚美兒童：「你好厲害！玩得很棒！」
  • 發送獎勵貼紙或小禮物

數據確認 checklist

  • ✅ 確認家長後測 TC-CHEXI 已完成送出（代號與前測一致）
  • ✅ 點擊遊戲結果頁「匯出 CSV」，儲存本次遊戲數據
  • ✅ 確認觀察紀錄表填寫完整
  • ✅ 感謝家長與兒童參與：「謝謝你今天來玩遊戲！」

後測提醒話術

「孩子已經玩完了，麻煩您再掃描一次牆上的 QR code，填寫同一份問卷。這次一樣請填入剛才的代號『C001』（依實際代號），這樣我們才能把您兩次的填答配對在一起。」
📊 階段七：數據分析與家長回饋（5~10 分鐘）

遊戲結束後，當場產出分析報告讓家長了解孩子的表現。

  • 1. 點擊遊戲結果頁的「匯出 CSV」，儲存本次遊戲數據檔案
  • 2. 開啟 訓練效果分析頁面（教師家長專區 → 訓練效果分析）
  • 3. 將剛才匯出的 CSV 檔案拖曳上傳（可一次選多個檔案）
  • 4. 系統自動產出圖表報告，包含：正確率趨勢、反應時間、辨別力（d'）等
  • 5. 向家長展示報告，簡要說明孩子的表現亮點
💡 可以對家長說：「這是孩子剛才玩遊戲的分析報告，我們可以看到他在每個關卡的表現。」
正確率趨勢各回合的答對比例，觀察是否逐步進步
反應時間按鈕速度，越快通常代表越熟悉規則
辨別力 (d')區分 Go 和 No-Go 的能力，數值越高代表抑制控制越好
工作記憶記住順序的正確率（若有玩到工作記憶關卡）
⚠️ 常見狀況處理
兒童不想玩／哭鬧 不勉強，先安撫情緒。可聊天或玩其他活動後再嘗試；仍拒絕則記錄後結束
遊戲卡住／當機 重新整理頁面（F5 / 下拉重新整理），遊戲資料會自動保存
兒童按錯位置 適當口頭提示按鈕位置，在紀錄中標註「需口頭提示」
家長想提示答案 請家長坐在視線外，說明：「讓孩子自己試試看效果最好」
中途想上廁所 暫停遊戲，記錄中斷情形，回來後可繼續或重新開始
觸控不靈敏 確認螢幕乾淨、手指乾燥，必要時改用觸控筆
